WebBlocks Kit

The WebBlocks Kit is a "starter kit" of HTML, CSS, Javascript, fonts and images that make it easy to build UCLA website and applications. The Kit is styled for the UCLA brand, and it is responsive-ready, meaning a site built with the Kit works on all devices from mobile to widescreen. The Kit also implements modern web standards, adds compatibility support for older browsers, provides user experience best practices,

Getting Started

Download the latest version of WebBlocks Kit from GitHub:
github.com/ucla/WebBlocks-kit/releases

Consult the design guide for patterns and code snippets:
spaces.ais.ucla.edu/display/webtoolkit/IMS+Web+Design+Guide

Using the Kit

There are several ways to use the Kit.

Directly from the Kit

This file already uses the Kit; therefore, using this file as a template, you can build your HTML files directly within this download.

In Your Own Applications

If you've already got an application, you can port the Kit directly into your application. To do this, copy the following folders into your application:

  1. css
  2. font
  3. img
  4. js

Once you've done this, then add the following code to the head section of your site:

<link rel="stylesheet" href="css/blocks.css">
<script type="text/javascript" src="js/blocks.js"></script>
<!--[if lte IE 9]>
<link rel="stylesheet" href="css/blocks-ie.css">
<script type="text/javascript" src="js/blocks-ie.js"></script>
<![endif]-->
<meta name="viewport" content="width=device-width, initial-scale=1">
